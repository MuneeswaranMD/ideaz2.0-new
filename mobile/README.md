# Averqon CRM Mobile App

This is the mobile companion for the Averqon CRM.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run the App**:
    ```bash
    npx expo start
    ```

## Features

-   **Authentication**: Login with your CRM credentials.
-   **Timesheets**: Clock In and Clock Out from your mobile device. Data syncs instantly with the web CRM.

## Configuration

The Firebase configuration is located in `firebaseConfig.js`. It shares the same database as the web application.
